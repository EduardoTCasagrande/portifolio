Meus Projetos

O projeto de Cadastro de Posts é uma aplicação web que permite aos administradores criar e excluir posts.

Utiliza as seguintes tecnologias:

Node.JS: Plataforma de execução de código JavaScript do lado do servidor.

Express.JS: Framework web para Node.js, facilitando a criação de APIs.

MariaDB: Sistema de gerenciamento de banco de dados relacional.

Handlebars: Engine de modelo para criar views dinâmicas no servidor

Css: Linguagem para Personalizar o site.

Além disso, o projeto inclui uma área de administração protegida por autenticação. O login de administrador e a senha proporcionam acesso exclusivo às funcionalidades de gerenciamento de posts.
