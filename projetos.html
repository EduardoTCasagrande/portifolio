O projeto de Cadastro de Posts é uma aplicação web que permite aos administradores criar e excluir posts.

Tecnologias Utilizadas:

Node.JS: Plataforma de execução de código JavaScript do lado do servidor.

Express.JS: Framework web para Node.js, facilitando a criação de APIs.

MariaDB: Sistema de gerenciamento de banco de dados relacional.

Handlebars: Engine de modelo para criar views dinâmicas no servidor

Css: Linguagem para Personalizar o site.

Além disso, o projeto inclui uma área de administração protegida por autenticação. O login de administrador e a senha proporcionam acesso exclusivo às funcionalidades de gerenciamento de posts.

O Sistema de Busca de Bandeiras é uma aplicação web desenvolvida com Node.js, Express, Handlebars e CSS. Este projeto permite aos usuários encontrar a bandeira de um país digitando o nome do país na barra de pesquisa.

Tecnologias Utilizadas:

Node.js e Express: Utilizados para criar o backend da aplicação e gerenciar as rotas. O Node.js possibilita a execução de JavaScript no servidor, enquanto o Express simplifica o desenvolvimento web.

Handlebars: Mecanismo de templates para gerar dinamicamente o conteúdo HTML. No projeto, o Handlebars é usado para renderizar a página com a bandeira correspondente ao país pesquisado.

CSS: Folhas de estilo para estilizar a aplicação e proporcionar uma experiência visual agradável.

Funcionalidade Principal:

Busca de Bandeiras: Os usuários podem digitar o nome de um estados na barra de pesquisa. Ao enviar o formulário, a aplicação retorna a bandeira correspondente do estado buscado.
